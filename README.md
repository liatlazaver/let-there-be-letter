# ויהי אות · ראש מילין

ממשק אינטראקטיבי לחקר האלף־בית העברי, בהשראת הספר *ראש מילין* מאת הרב אברהם יצחק הכהן קוק.
כל אות נבנית לנגד עיניך מאותיות היסוד שלה באנימציית מורף, ואפשר לפרק כל אות לרכיביה,
לקרוא על משמעותה, ולנווט בין האותיות, הניקוד והתגין.

פרויקט גמר בתקשורת חזותית — מאת ליאת לזובר.

## הפעלה מקומית

זהו אתר סטטי (ללא שרת). כדי להריץ אותו במחשב, מתוך תיקיית הפרויקט:

```bash
python3 -m http.server 8000
```

ואז לפתוח בדפדפן: <http://localhost:8000>

## מבנה

- `index.html` — נקודת הכניסה
- `v12.js` — המודול הראשי (טוען את שאר החלקים)
- `assets/` — האותיות, התמונות, ואנימציות המורף
- מסכי האותיות, הניקוד והאודות — כל אחד במודול משלו
